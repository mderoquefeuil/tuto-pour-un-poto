POUR LE CSS

LES POLICES

Les polices à utiliser sans risque sont :

  • Arial
  • Arial Black
  • Comic sans MS
  • Courier new
  • Georgia
  • Impact
  • Times New Roman
  • Trebuchet MS
  • Verdana
Ces polices sont reconnues par la plupart des navigateurs.

Si on veut utiliser une autre police, on peut utiliser ce site pour en télécharger.

LES COULEURS

on peut ecrir les couleurs de 3 façons de la plus simple à la plus compliquée:

  1. avec le nom de la couleurs en anglais
  2. avec le code exadecimal (ex: #FFFFFF)
  3. avec la notation rgb(xxx,xxx,xx)

Pour trouver le code exadécimal d'une couleur, on peut rechercher color picker sur google

LES BORDURES

On peut mettre des bordures avec la notation "border: "
Il existe plusieurs type de bordures

Par défaut la bordure entoure tout le texte sinon on peut choisir de la faire apparaitre d'un coté seulement en utilisant une des bordures ci dessous :

  • border-top
  • border-right
  • border-left
  • border-bottom

On peut faire des arrondis avec border-radius.
Par exemple MARTIN EST UN DON JUAN

On peut aussi ajouter de l'ombre avec border-shadow en indiquant dans l'ordre : la verticale, l'horizontal, l'adoucissement et la couleur. Ca marche aussi pour le texte.
Par exemple : J'AI UN GROS ZIZI

APPARENCES DYNAMIQUES

on peut ajouter une comande de survol qui s'appelle :over, par exemple pour appliquer au paragraphe, on peut ecrire: "p:hover{}"

on peut trouver plus de details sur la partie du cours dédié aux apparences dynamiques sur openclassroom
